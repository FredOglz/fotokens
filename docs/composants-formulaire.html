4 · Composants — formulaire

Button, Input, Textarea, Select, Checkbox, Radio, Switch. Rendus uniquement avec les variables générées.

Rien n'est écrit à la main. Seul le button est role-aware : il est généré pour les 6 rôles à partir d'une table qui dit quels slots chaque variante consomme (<Button color="danger"> existe sans un token de plus). checkbox, radio et switch, eux, sont mono-parti : une case n'a pas de couleur à choisir, elle est primary. Ils gardent en revanche un état d'erreur qui pointe le rôle danger — comme le error d'un champ.
