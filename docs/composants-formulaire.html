4 · Composants — formulaire

Button, Input, Textarea, Select, Checkbox, Radio, Switch. Rendus uniquement avec les variables générées.

Rien n'est écrit à la main. Les composants role-aware (button, checkbox, radio, switch) sont générés pour les 6 rôles à partir d'une table qui dit seulement quels slots chaque variante consomme. C'est la promesse du contrat de rôle : <Checkbox color="danger"> existe sans un token de plus.
