3 · Sémantique (le reste)

Au-delà du contrat de rôle : les fonds, le texte, les bordures, le focus, les tailles et la typo.

Couleur

surface est le seul groupe qui parle de fonds.

surface

base
Fond de l’application.
raised
Fond d’un bloc surélevé (carte, panneau).
overlay
Fond d’un élément flottant (popover, menu, modale).
sunken
Fond en creux (piste de slider, zone de dépôt).
inverse
Rupture visuelle négative (tooltip, bandeau).
disabled
Fond d’un composant désactivé.
transparent
Fond absent (variantes outlined/ghost).

text

strong
Titres et libellés d’interface appuyés. Le contraste maximal.
body
Texte de lecture.
muted
Texte secondaire (légende, description).
faint
⚠ NE PASSE PAS AA — décoratif uniquement (séparateurs, échafaudage). Jamais pour du texte à lire.
disabled
Texte d’un composant désactivé.
inverse
Texte posé sur `surface.inverse`.

border

default
Bordure par défaut de tous les composants.
strong
Bordure appuyée.
divider
Séparateur entre deux blocs de contenu.
disabled
Bordure d’un composant désactivé.
transparent
Bordure absente.

Focus & scrim

focus.ring
Anneau de focus clavier. Doit rester visible sur tous les fonds.
scrim
Voile assombrissant sous une modale ou un drawer.

Breakpoints. Une media query CSS ne peut pas lire une custom property — @media (max-width: var(--size-breakpoint-md)) ne fonctionne pas. Le build lit donc size.breakpoint.md dans les tokens et inline sa valeur. Une seule source de vérité.
